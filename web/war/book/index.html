Flexible User Interface
[size=100]The user interface of GeoGebra is flexible and can be adapted to the needs of your students. [br][br]If you want to use GeoGebra in early middle school, you might want to work with a blank sheet in the [img]http://wiki.geogebra.org/uploads/thumb/c/c8/Menu_view_graphics.svg/16px-Menu_view_graphics.svg.png[/img] [i]Graphics View[/i] and geometry tools. Later on, you might want to introduce the [img]http://wiki.geogebra.org/uploads/thumb/7/7d/Stylingbar_graphicsview_show_or_hide_the_axes.svg/16px-Stylingbar_graphicsview_show_or_hide_the_axes.svg.png[/img] [i]coordinate system[/i] using a [img]http://wiki.geogebra.org/uploads/thumb/f/f1/Stylingbar_graphicsview_show_or_hide_the_grid.svg/16px-Stylingbar_graphicsview_show_or_hide_the_grid.svg.png[/img] [i]grid[/i] to facilitate working with integer coordinates. [br][br]In high school, you might want to use algebraic input in order to guide your students through algebra on into calculus.[/size]

[size=100]Each GeoGebra [i]View [/i]provides a [i]Toolbar [/i]with a selection of [i]Tools [/i]specific for the [i]View [/i]you are working with. What are Toolboxes?
[size=100]GeoGebra's [i]Tools[/i] are organized in [i]Toolboxes[/i], containing similar [i]Tools [/i]or [i]Tools [/i]that generate the same type of new object. You can open a [i]Toolbox [/i]by clicking on the lower part of a [i]Tool [/i]button and selecting a [i]Tool [/i]from the appearing list.[br][br][b]Task[/b]: Find the [icon]GeoGebra/images/ggb/toolbar/mode_segment.png[/icon] [i]Segment[/i] tool within the [i]Toolboxes[/i][br][/size][list=1][*][size=100]Search the [i]Toolboxes [/i]for the [i]Segment [/i]tool[br][/size][/*][*][size=100]Click in the [i]Graphics View[/i] twice in order to create two points and a segment between them.[/size][/*][*][size=100]Explain why you need to create two points and find out whether the order of their creation matters for this tool.[/size][/*][/list][size=100][br][/size]

3.3.

Tips and Tricks for Algebraic Input

Confirm an Expression
[size=100]Always confirm the expression you entered into the[i] Input Field [/i]by pressing the [i]Enter[/i] key.[/size]
Name a New Object
[size=100]Type [font=Courier New]name =[/font] into the [i]Input Field [/i]in front of its algebraic representation in order to name a new object.[br][u]Example[/u]: [font=Courier New]P = (3, 2)[/font] creates point [i]P[/i].[/size]
Indicate Multiplication
[size=100]Multiplication needs to be entered using an asterisk or space between the factors.[br][u]Example[/u]: Type in either [font=Courier New]a*x[/font] or [/size][font=Courier New][size=100]a x[/size][/font]
Upper and Lower Case Letters
[size=100][b]GeoGebra is case sensitive![/b] Thus, upper and lower case letters must not be mixed up.[br][br][u]Note[/u]:﻿[br][list][*]Points are always named with upper case letters.[br][u]Example[/u]: [font=Courier New]A = (1, 2)[/font][br][/*][*]Vectors are named with lower case letters.[br][u]Example[/u]: [font=Courier New]v = (1, 3[/font])[br][/*][*]Segments, lines, circles, functions... are always named with lower case letters.[br][u]Example[/u]: circle [font=Courier New]c: (x – 2)^2 + (y – 1)^2 = 16[code][br][/code][/font][/*][*]The variable [i]x[/i] within a function and the variables [i]x[/i] and [i]y[/i] in the equation of a conic section always need to be lower case.[br][u]Example[/u]: [font=Courier New]f(x) = 3*x + 2[br][/font][/*][/list][/size]
Try it Out...
Use an Object within an Algebraic Expression
[size=100]If you want to use an object within an algebraic expression or command you need to create the object prior to using its name in the [i]Input field[/i].[br][br][u]Examples[/u]:[/size][list][*][size=100]The input [font=Courier New]y = m x + b[/font] creates a line whose parameters are already existing values [i]m [/i]and [i]b [/i](e.g. numbers / sliders).[/size][/*][*][size=100]The input [font=Courier New]Line[A, B][/font] creates a line through existing points [i]A[/i] and [i]B[/i].[/size][/*][/list]
Error Messages
[size=100]Always read the error messages – they could possibly help you to fix the problem![/size]

Excursion into Physics
[size=100]Sound waves can be mathematically represented as a combination of sine waves. Every musical tone is composed of several sine waves of form [i]y(t) = a sine(ω t + φ)[/i].[br][br]The amplitude [i]a[/i] influences the volume of the tone while the angular frequency [i]ω[/i] determines the pitch of the tone. The parameter [i]φ[/i] is called "phase" and indicates if the sound wave is shifted in time.[br][br]If two sine waves interfere, superposition occurs. This means that the sine waves amplify or diminish each other. We can simulate this phenomenon with GeoGebra in order to examine special cases that also occur in nature.[/size]
