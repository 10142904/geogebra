GeoGebra - Dynamic Mathematics for Everyone
http://www.geogebra.org/

This directory contains the "desktop" and the "web" version of GeoGebra.
The "common" directory contains the common parts for the two interfaces.

For compiling, running and installing the "desktop" version, please read
http://www.geogebra.org/trac/wiki/SetUp.

To compile and run the "web" version (a.k.a. GeoGebraMobile), please
read http://www.geogebra.org/trac/wiki/GeoGebraMobile.
